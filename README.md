# CustomerReviewSummarizeSystem

一个简答的用户情感分析系统 ,使用的是开源数据   [Yelp DataSet](https://www.yelp.com/dataset/documentation/main) ，对标国内的大众点评。此项目仅用到 business.json 和 review.json 两个文件，下载解压后放入到data文件中。



## Run



**1、Text processing**

```python
 python data.py
```



**2、Training  model**

```python
 python main.py
```

## Test

 python test.py
